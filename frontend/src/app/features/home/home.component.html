Exchange Rate System

Spread-adjusted currency rates, historical trends, usage analytics, and AI-powered insights. Demo data covers the last 30 days for all supported currencies below.

How to use

1. Calculator

Pick a from and to currency (3-letter ISO code). Optionally choose a date with the calendar — leave empty for the latest available rate. The API returns a spread-adjusted exchange rate and query counters.

Open Calculator

2. Historical

Select a currency pair and a date range (calendar pickers). View a table, line chart, and an AI trend summary for the period. Try EUR → PLN or EUR → GBP over the last month.

Open Historical

3. Analytics

See which currencies are queried most often. Each lookup from the Calculator increments usage counters for both currencies in the pair.

Open Analytics
